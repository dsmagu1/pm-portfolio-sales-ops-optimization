# Sales Performance Optimization Initiative  
*Fortune 100 Insurance Company (Anonymized)*

## Overview
This repository documents a sales operations optimization initiative conducted within a high-volume inbound insurance sales environment. The project focused on improving visibility into performance drivers, strengthening feedback loops between operational changes and outcomes, and supporting leadership decision-making through structured KPI tracking and reporting.

All materials are anonymized and reconstructed to demonstrate project leadership, delivery approach, and operational decision-making without exposing proprietary or confidential information.

---

## Business Problem
Sales leadership lacked consistent visibility into how operational and strategic changes impacted frontline performance. Performance fluctuated across agents and shifts, and feedback loops between data, leadership decisions, and execution were slow.

Key challenges included:
- Limited clarity around KPI definitions and ownership
- Inconsistent performance after strategy changes
- Delayed insight into what was working vs. not working
- Need for structured reporting to support decision-making

---

## Project Goals & Success Metrics
**Primary goals**
- Establish a consistent KPI framework for sales performance
- Improve clarity around performance trends following operational changes
- Enable faster, data-informed adjustments
- Support leadership with actionable insights

**Key metrics (directional, anonymized)**
- Conversion rate
- Productivity / throughput
- Schedule adherence
- Trend stability following changes
